# BNC_WOA
The source code of method proposed in the paper "Predicting Spatial Distribution of Oil and Gas Resources Using a Discrete Whale Optimization-Enhanced Bayesian Network Classifier"

Instructions for Use
1. Install the BNT Toolbox: https://github.com/bayesnet/bnt
2. Run main.m. This file includes the process of reading sample data, constructing the BNC_WOA network, classifying test data, and displaying evaluation metrics such as accuracy and recall.
Note: It should be noted that due to the confidentiality requirements of our data, we resampled 300 instances for readers to test our code.
